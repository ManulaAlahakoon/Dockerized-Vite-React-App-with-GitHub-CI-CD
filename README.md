<h1>CI/CD Pipeline for Multi-Stage Docker Build and Docker Hub Deployment</h1>
<h2>Technologies: Docker, GitHub Actions, Docker Hub, React (Vite)</h2>

<p>Built a containerized React (Vite) application using a multi-stage Docker build for optimized production images</p>
<p>Implemented a CI/CD pipeline using GitHub Actions to automatically build and push Docker images to Docker Hub on every commit</p>
<p>Used versioned tagging (e.g., v-1, v-2) for consistent and traceable releases</p>
